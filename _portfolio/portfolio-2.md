## 🎮 Obstacle Dodging Game (Web Game)

[![Play Game](https://img.shields.io/badge/PLAY-GAME-0A84FF?style=for-the-badge&logo=gamepad&logoColor=white)](https://hamsdgfh.github.io/java_25.11.20/)
[![GitHub Repo](https://img.shields.io/badge/GitHub-Repository-181717?style=for-the-badge&logo=github&logoColor=white)](https://github.com/hamsdgfh/java_25.11.20)

---

🚀 **프로젝트 개요**  
AI의 도움을 받아 제작한 **브라우저 기반 장애물 피하기 게임**입니다.  
HTML·CSS·JavaScript를 사용해 기획 → UI → 게임 루프 → 충돌 판정까지 전 과정의 구조를 직접 설계했습니다.  
난이도 선택, 장애물/아이템 스폰, 점수·체력 시스템 등 기본적인 웹게임 요소들을 구현하며 게임 개발의 핵심 로직을 경험한 프로젝트입니다.

---

🛠️ **사용 기술 및 환경**  
- **Languages:** HTML, CSS, JavaScript  
- **Game Loop:** `requestAnimationFrame` 기반  
- **Storage:** `localStorage`를 통한 최고 점수 저장  
- **Deployment:** GitHub Pages  
- **Version Control:** Git  

---

✨ **주요 구현 내용**  
- **난이도 선택 (Easy / Normal / Hard)**  
- **장애물 및 아이템 자동 스폰 시스템**  
- **충돌 감지 로직 및 피격 처리**  
- **체력(Heart) & 보호막(Shield) 시스템 구현**  
- **HUD 구성:** 점수, 난이도, 체력 UI 표시  
- **네온 스타일 UI 디자인 & 키보드 컨트롤(← →)**  

---

🧩 **프로젝트 특징**  
- 브라우저만으로 실행 가능한 **경량 웹게임**  
- 충돌 판정, 게임 루프, 스폰 주기 설계 등 **게임 개발 기본 구조 직접 구현**  
- 인터벌 중첩 문제 개선 및 구조 최적화 경험  
- 단순한 룰이지만 **동작 구조·렌더링·상태 관리 학습에 적합**

---

📝 **배운 점**  
- `requestAnimationFrame`을 활용한 **게임 루프 구조 이해**  
- 장애물·아이템 스폰, 난이도 조절 등 **상태 변화 기반 로직 설계**  
- localStorage를 통한 간단한 저장 기능 구현  
- AI를 이용한 디버깅 및 성능 개선 과정 경험  
- CSS로 UI 스타일링과 HUD 구성 능력 향상
